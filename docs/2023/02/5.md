## 题目链接 [P1028 [NOIP2001 普及组] 数的计算](https://www.luogu.com.cn/problem/P1028)

## 完整代码
```cpp
#include <iostream>
using namespace std;
int main(void){
    int e[1111];
    int t;
    cin>>t;
    e[1]=1;
    for(int i=2;i<=t;i++){
        e[i]=e[i-1];
        if(i%2==0){
            e[i]+=e[i/2];
        }
    }
    cout<<e[t];
    return 0;
} 
```
